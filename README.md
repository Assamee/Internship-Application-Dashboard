# Internship Application Dashboard

[🚀 Live Demo](https://your-custom-url.vercel.app/)

## ✨ Features

* **Persistent Storage:** Utilises the browser's `localStorage` API to save application data and user theme preferences across page refreshes.
* **Dark/Light Mode:** Features a customisable Dark/Light mode toggle that instantly updates CSS variables across the entire application.
* **Full CRUD Functionality:** Users can Create, Read, Update, and Delete job applications using state-driven modal forms.
* **Real-time Search & Filtering:** A search bar  to allow users to instantly filter their applications by either 'Company' or 'Job Role'.
* **Visual Hierarchy:** Uses React-Bootstrap Badges to visually categorise priority levels and application statuses (e.g., Online Assessment, Interview, Offer).
* **Smart Clipboard Integration:** Includes a dedicated "View" modal for reading lengthy job descriptions, complete with a one-click copy-to-clipboard function.
* **Responsive Grid Architecture:** Built with Bootstrap's 12-column grid system, ensuring the dashboard is fully usable on mobile phones, tablets, and ultrawide desktop monitors.

## 🛠️ Technologies Used

* **Frontend Framework:** React.js (Hooks: `useState`, `useEffect`)
* **UI Component Library:** React-Bootstrap
* **Version Control:** Git & GitHub

## 🚀 Getting Started

To run this project locally on your machine, follow these steps:

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your computer.

### Installation

1. Clone the repository:

    ```bash
    git clone https://github.com/Assamee/Internship-Application-Dashboard.git
    ```

2. Navigate into the project directory:

    ``` bash
    cd Internship-Application-Dashboard
    ```

3. Install the required dependencies:

    ``` bash
    npm install
    ```

4. Start the development server:

    ``` bash
    npm run dev
    ```

5. Open your browser and navigate to `http://localhost:5173` to view the application.
